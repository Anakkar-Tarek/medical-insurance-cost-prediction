# Dataset Information

## Medical Insurance Cost Dataset

**File:** `insurance.csv`

### Description
This dataset contains medical insurance cost information for beneficiaries.

### Features
- `age`: Age of primary beneficiary
- `sex`: Gender (male/female)
- `bmi`: Body Mass Index
- `children`: Number of dependents
- `smoker`: Smoking status (yes/no)
- `region`: US region (northeast, southeast, southwest, northwest)
- `charges`: Medical costs billed (TARGET VARIABLE)

### Statistics
- Total records: 1,338
- File size: 55KB
- No missing values
